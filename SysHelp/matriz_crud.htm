Matriz CRUD
 
CRUD es el acrónimo de en inglés: Create, Read, Update y Delete.
Se usa para referirse a las funciones básicas de operaciones hacia las tablas de base de datos.
 
Orientada a listar los componentes que Utilizan la TABLA base. 
 
Características:
 
  •  
     
    Muestra para cada componente el modo en que se utiliza: Select, Insert, Update, Delete.
  •  
     
    Es posible presentar visualmente el componente fuente donde se está haciendo uso de la tabla.
 
Este producto está disponible en todas las plataformas, siempre y cuando se use Base De Datos.

En esta ventana se puede especificar como condición el nombre del Sistema y el nombre de una tabla de la cual se podrá obtener la matriz CRUD, es decir, el tipo de operación que se realiza con ella como puede ser crear ó insertar, actualizar ó borrar un registro de la tabla.
Si en la condición donde se especifica el nombre de la tabla se indica un asterisco la información mencionada anteriormente,  se obtendrá para todas las tablas.
En todos los casos se puede imprimir ó exportar a Excel.
 
Cuando se elige alguno de los componentes de la matriz CRUD y dando doble clic aparece una ventana flotante o menú contextual a partir del cual se puede solicitar: análisis de impacto, diagrama de proceso, documentación, lista de componentes, propiedades, referencias cruzadas,  ver fuente ó versiones.
